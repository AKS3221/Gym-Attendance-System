PROJECT TITLE:
Gym Management System

PURPOSE OF PROJECT:
The purpose of this project is to develop a Gym Management System using Java.
This system helps manage gym members by storing and handling member details,
including regular and premium memberships.

The project provides a graphical user interface (GUI) that allows users to:
- Add new gym members
- Manage regular and premium member information
- Activate or deactivate memberships
- Track attendance and membership details
- Handle premium membership payments and discounts
- Display and manage member records efficiently

This project demonstrates the use of Java programming concepts such as:
Object-Oriented Programming (OOP), inheritance, abstraction, ArrayList,
GUI development using Java Swing, and event handling.

HOW TO START THIS PROJECT:
1. Install Java Development Kit (JDK) on your computer.
2. Open the project folder in a Java IDE such as:
   - IntelliJ IDEA
   - Eclipse
   - NetBeans

3. Compile all Java files:
   - GymGUI.java
   - GymMember.java
   - RegularMember.java
   - PremiumMember.java

4. Run the GymGUI.java file.

5. The Gym Management System window will open and the user can start
   managing gym members.
